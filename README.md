# Simple Weather App

一个使用 OpenWeatherMap 接口的纯前端天气查询小应用。输入城市名称即可获取当前温度、天气图标与描述，支持重复城市检测与国家代码（如 `shenzhen,cn`）的精确匹配。

## 功能特性

- 城市天气查询（摄氏度）
- 天气图标与文字描述展示
- 重复城市提示与国家代码支持
- 响应式布局，适配不同屏幕

## 项目结构

```
weather/
├─ index.html        # 页面结构与入口
├─ css/
│  └─ style.css      # 页面样式
└─ js/
   └─ main.js        # 交互逻辑与数据获取
```

## 快速开始

直接双击打开  
   - 将本项目下载到本地后，双击 `index.html` 用浏览器打开即可使用。

## 配置 API Key

本项目使用 OpenWeatherMap 的公开接口。请按以下步骤获取并配置密钥：

1) 前往 https://home.openweathermap.org/api_keys 申请并复制你的 API Key  
2) 打开 [main.js](./js/main.js)，找到如下位置：
   ```js
   const apiKey = "你的API密钥";
   ```
3) 将其替换为你自己的密钥后保存即可

> 提示：出于安全考虑，不建议在公开仓库中提交真实密钥。正式部署时可改为通过后端代理请求或在服务器侧注入密钥，避免在前端代码中暴露。

## 使用说明

- 在输入框中输入城市名（可选：加上国家代码，如 `paris,fr`），点击 SUBMIT 即可查看天气  
- 若该城市已在列表中，页面会提示避免重复添加  
- 输入无效城市名时，会显示错误提示
